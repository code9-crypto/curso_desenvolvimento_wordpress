OPÇÕES DE PERSONALIZAR DO TEMA

(Editar / Cadastrar um produto novo:
	Na seção do OceanWP Configurações ->
		Título(title) -> Esta opção desativa o título da página(essa parte fica abaixo do cabeçalho)
		Caminho de navegação(Breadcrumbs) -> Esta opção é possível desativar o caminho onde você está localizado

Tipografia -> Personaliza as fontes para o corpo do texto quanto para os títulos

Estilo e configurações do site ->
	Rolagem para o topo -> Aqui é possível ativar/desativar ou fazer configurações no botão de rolar para cima

Páginas e páginas especiais ->
	> Layout da página -> Onde define o layout de todas as páginas(exceto a de produtos/loja)

Barra Superior -> Opção de ativar ou desativar barra superior

Cabeçalho -> Onde é possível fazer configurações de cores, bordas, tipos, espaçamentos do cabeçalho
	Geral > Cores -> Define cores do fundo do cabeçalho e da borda inferior
	Geral >Tipo de cabeçalho -> Opções de layout do cabeçalho
	Logo e media -> Alterar imagens de mídia e logo
		Logo -> Altera a imagem e também o tamanho(altura e largura) da imagem
	Menus -> Onde é possível personalizar os menus(MAS NÃO CRIA-LOS)
		Posição do menu -> define como será exibido: ESQUERDA, CENTRO , DIREITA
		Efeitos do link -> Quando o mouse passa por cima dos links, um efeito aparece
		Links -> Define cores tanto fixo quanto ao passar o mouse
		Pesquisa no cabeçalho -> Configura cores e forma de como a pesquisa é mostrada

Widgets do rodapé -> Onde é possível fazer configuração da quantidade de colunas, espaçamentos, cores ...

Direitos autorais no rodapé -> Onde é possível fazer configuração de espaçamentos, alterar o texto dos direitos autoriais...


WooCommerce -> Configura itens a respeito das vendas(loja, minha conta, página de pagamento...)
	Geral -> Onde é possível ativar/desativar a barra lateral do woocommerce e fazer mais alguns ajustes
	Menu do carrinho -> Onde é possível fazer a configuração do ícone(imagem) e visibilidade
	Ícone do menu -> Onde é possível alterar o ícone do carrinho
		Estilo do ícone do menu -> Onde é possível trocar o ícone, tamanho e posição vertical
	Arquivos ->
		> Layout -> Onde é definido o layout de como os produtos serão exibidos de forma agrupadas	
	Produto único ->
		> Layout -> Onde define o layout da exibição dos produtos de forma individual	

(ESTES ITENS POR PADRÃO ESTÃO NA MAIORIA DOS TEMAS)
Identidade do site -> Onde é definido: título do site, descrição e o ícone
Menus -> Configura os menus
Widgets -> Configura os widgets, de acordo com cada página(lateral, rodapé etc...)
Configuração da página inicial -> onde é configurado a página inicial e a página de blog
