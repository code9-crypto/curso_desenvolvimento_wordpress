WORDPRESS
Projeto(10 minutos de exercícios) com tema sydnei

Em "personalizar" vai abrir uma tela com várias opções:
Geral->Layouts => 
	Invólucro da Página - Espaçamento superior = Configura o espaçamento da parte superior
	Invólucro da Página - Espaçamento inferior = Configura o espaçamento da parte inferior

Cabeçalho->Identidade do site => É possível configurar: título, subtítulo, ícone da aba, imagem de logo ...
Cabeçalho->Main header
					->geral => É possível configurar a posição do menu, habilitar o sticky e o tipo do Container(Contained ou Full-width)
					->style => Possível configurar: cor de fundo do menu/submenu, tamanho da fonte ...
Cabeçalho->Mobile Header
					->geral => É possível configurar o layout do menu, a forma que o offcanvas aparece(ícone com 3 traços no canto superior esquerdo ou direito),
					alinhamento dos links no menu, separador de link e no offcanvas->Tipo de cabeçalho é possível configurar como: marca do site(site branding), nada(nothing) ou Custom text(texto customizável)
					->style => pode configurar a estilização da barra de menu, offcanvas menu e link separator

Hero area->hero type =>
	Tipo de cabeçalho da primeira página = Esta configuração será feita apenas para primeira página, podendo ser: Slider de tela cheia, imagem, video e sem cabeçalho(somente menu)
	Tipo de cabeçalho do site = Esta configuração será feita para as de mais páginas do site, podendo ser: Slider de tela cheia, imagem, video e sem cabeçalho(somente menu), exceto a primeira página
Hero area->hero slider => De forma bem simples e intuitiva de selecionar as imagens, o tempo para passar e configurar o botão
Hero area->Mídia do cabeçalho => onde é possível configurar um vídeo no topo do site(fica abaixo do menu)

Typography -> Define fonte, tamanho e tipo de fonte tanto do body(corpo) quanto do headings(título dos textos)

Pages -> Define o layout das páginas(inclui a página inicial), são esses: Normal, Narrow(Encaixotada) e Stretched(esticado)
OBS.: a opção que habilita o sidebar, permite que seja inserido widgets na lateral(na própria opção Wigets)

Blog
Blog Archives -> Define como é definido o layout dos posts, configura cada item do post(imagem(alinhamento), resumo do post(alinhamento do texto), data da publicação) e a lateral(que é side bar)
Single posts -> semelhante a Pages, porém é SOMENTE aos posts(exibição individual)

Rodapé->Footer widgets => Configurar quantidade de colunas e também cores(de fundo e do texto), e também permite configurar a quantidade de widgets
Rodapé->Footer copyright area => Configurar cores(fundo e texto) e configurar o texto que vai aparecer

Cores => Nela é possível configurar todas a cores do site: cabeçalho, rodapé, corpo da página ...
	Links -> Nesta seção é possível configurar as cores do botão "VEJA MAIS  e do botão SCROLL TO TOP"


Menus -> Onde é possível criar o menu e configurar a posição de cada item

Widgets -> É opção que configura os itens(texto, imagem, etc...) que vão aparecer no rodapé, lateral ou onde a página suportar

Configurações da página inicial => nela é possível configurar qual será a página inicial e qual será a página de posts
